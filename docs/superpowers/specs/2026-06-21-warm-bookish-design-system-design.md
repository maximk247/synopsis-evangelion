# Дизайн: тёплая книжная дизайн-система (по образцу read_kings)

Дата: 2026-06-21
Область: apps/web (SvelteKit, prerender, adapter-static)
Статус: согласовано, готово к плану реализации

## Контекст

apps/web - читательское приложение "Евангельский синопсис": оглавление перикоп
с индикатором присутствия в 4 евангелиях, режим сплошного чтения и ключевой
экран параллельных колонок синопсиса. Текущая дизайн-система (app.css) - нейтральная,
с 3 темами (light/sepia/dark), переключателем засечек и размером шрифта.

Эталон направления - соседний проект d:/read_kings (Next.js): тёплая книжная
палитра amber/stone, шрифты Inter (UI) + Alegreya (serif), выбор шрифта чтения
из выпадающего списка с сохранением в localStorage.

Незакоммиченная WIP-переработка дизайна отложена в git stash
("wip-design-rework-2026-06-21 (pre fresh redesign)"); новая работа стартует
с чистой базы коммита.

## Цели

1. Перевести всю дизайн-систему в тёплый книжный вид по образцу read_kings.
2. Сократить темы до двух: светлая кремовая и тёмная stone (sepia убрать).
3. Добавить выбор шрифта чтения из списка (как в read_kings), дефолт Calibri.
4. Полный редизайн: токены, общие примитивы, перекомпоновка экранов.

## Решения (согласованы)

- Старт со свежего направления; WIP в stash.
- Настроение: тёплое книжное, палитра read_kings 1:1.
- Объём: полный редизайн (токены + примитивы + экраны).
- Темы: 2 (light кремовая, dark stone), sepia удаляется.
- Шрифт чтения по умолчанию: Calibri; выбор из списка сохраняется.
- Шрифты подключаются self-host через @fontsource (офлайн, кириллица).

## Типографика и выбор шрифта

### Подключаемые шрифты (self-host, @fontsource)

- Inter (--font-ui): интерфейс - шапка, навигация, кнопки, бейджи, заголовки разделов.
  Пакет @fontsource-variable/inter, импорт кириллического сабсета.
- Alegreya (--font-serif): книжные заголовки и один из вариантов шрифта чтения.
  Пакет @fontsource-variable/alegreya (cyrillic).

Импорт в app.css или в +layout, чтобы шрифты попадали в бандл и работали офлайн.

### Шрифт чтения

- Новая CSS-переменная --reading-font на documentElement, применяется к тексту
  евангелий: классы .verse-text (и алиас .bible-text), колонки синопсиса
  (PericopeColumns), режим чтения (/read/[g]).
- Параметры текста чтения: line-height 1.85, letter-spacing 0.01em.
- Выпадающий список из 15 вариантов (1:1 с read_kings):
  Alegreya, Tahoma/Geneva/Verdana, Georgia, Times New Roman, Palatino, Garamond,
  Cambria, Bookman, Trebuchet MS, Verdana, Segoe UI, Arial, Lucida Sans,
  Calibri (по умолчанию), Courier New.
- Каждое значение - стек с фолбэком на serif/sans, чтобы кириллица всегда
  отрисовывалась (часть системных шрифтов вроде Garamond/Bookman на Windows
  без кириллицы подставит фолбэк - это ожидаемо и приемлемо).
- Существующий тумблер "С засечками" удаляется: его роль поглощает выбор шрифта.
- Размер шрифта (A-/A/A+) сохраняется без изменений.

## Стор настроек (settings.svelte.ts)

- Тип Theme: 'light' | 'dark' (удалить 'sepia').
- Удалить поле serif и метод setSerif; удалить применение data-serif.
- Добавить поле readingFont: TReadingFontKey и метод setReadingFont.
- В apply(): выставлять --reading-font на documentElement по выбранному ключу;
  оставить data-theme и data-font.
- Persist: добавить readingFont в сериализацию; миграция старого значения
  theme==='sepia' в 'light' при чтении из localStorage.
- Константы шрифтов (ключ, label, value-стек) и дефолтный ключ вынести в
  отдельный модуль ($lib/data/fonts.ts) по образцу read_kings constants/fonts.ts.

## Примитивы и экраны

### Общие примитивы (на токенах)

- Кнопки/табы/переключатели (.switch в /read, ColumnTabs, кнопки в drawer):
  фон --card, граница --border, активное состояние --accent-wash + --accent,
  ховер --hover, мягкий радиус.
- Поля ввода (SearchBox, фильтр на главной): фон --card, фокус-кольцо янтарём.
- Карточки/строки (пункты оглавления, закладки, сноски, предисловия): на
  --card / --bg-soft с тёплой границей.
- Бейджи: GospelPresence (присутствие в евангелиях) и RefLinkBadge - янтарные
  токены вместо прежних синих.

### Экраны

1. Шапка (+layout.svelte): бренд-марка в янтарь/золото; ссылки навигации и
   кнопка настроек на тёплых токенах; backdrop-blur сохраняется.
2. Главная / Содержание (+page.svelte): секции и список перикоп в книжном виде,
   номер перикопы янтарём, GospelPresence справа; блок "Продолжить чтение" -
   тёплая карточка.
3. Режим чтения (/read/[g]): текст шрифтом чтения с line-height 1.85;
   переключатель евангелий и маркеры перикоп/глав на янтаре; номера стихов
   приглушённым золотом.
4. Синопсис / колонки (/p/[id]): заголовки колонок с янтарной нижней границей;
   подсветка параллельных строк --active-verse; мобильные табы в новом стиле.
5. Drawer настроек (SettingsDrawer.svelte): Тема (2 кнопки) - Размер шрифта -
   Шрифт (выпадающий список) - Подсветка параллелей.
6. Второстепенные экраны (bookmarks, prefaces, footnotes, appendix, search):
   подхватывают токены, лёгкая шлифовка карточек и строк.
